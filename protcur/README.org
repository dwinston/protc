* Setup
** Python
=pip install protcur=
** Gentoo
As root.
#+BEGIN_SRC bash
layman -a tgbugs-overlay
emerge -p protcur
vim /etc/portage/package.accept_keywords
emerge protcur
vim /etc/conf.d/protcur
rc-config add protcur default
/etc/init.d/protcur start
#+END_SRC
